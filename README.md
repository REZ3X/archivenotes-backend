## Description

This is the backend service for the ArchiveNotes application, built using the [Nest](https://github.com/nestjs/nest) framework. It provides APIs for creating, updating, deleting, and retrieving notes.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
